# 概要
- おすすめされたコンテンツ（動画や音楽、ゲームなど）を保存しておけるツール。
# 作成したきっかけ
- XなどのSNSで見つけたものや友人などからおすすめしてもらったコンテンツを一元的に管理する場所が欲しかった。
- また、おすすめしてくれた人を保存することで、その人と話すときに感想を話題にあげることができ、コミュニケーションの助けになると思った。
- 加えて、基本情報技術者試験や講義で学んだMVCモデルやwebアプリケーションの構造を実際に作ることで、自分の体験として知識を咀嚼できると考えた。
  - まだ講義でSpring等を学んでおらず、プログラミング言語の構文や使い方などは学んだもののそれをどう使うか、何を作るかの部分が実体として得られていなかったため、実際にこう使われているという事を理解したかった。
# 機能
## 実装済
- タイトル/作者/URL/おすすめしてくれた人を保存して一覧で表示
- 保存されたコンテンツにコメント、5段階評価の付与
- コンテンツの削除
## 未実装
- 認証機能（ログイン）
- アプリケーションサーバーを落としてもデータが保存される機能
  - 現在H2DatabaseのインメモリDBを利用しているため。MySQLへの移行を考えています。
- タイトルや教えてくれた人順のソート
- 評価のないものや評価済のもの、教えてくれた人などをキーとしたフィルタ機能
  - これは比較的実装が簡単だと思うので早めに追加したい。
